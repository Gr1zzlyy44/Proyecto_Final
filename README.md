# Proyecto Final – Aplicación Móvil

Aplicación móvil desarrollada como proyecto final, utilizando **React Native** y **Expo**.  
El objetivo del proyecto es aplicar conceptos de desarrollo móvil, organización de componentes y manejo de datos.

##  Tecnologías utilizadas
- React Native
- Expo
- JavaScript / TypeScript
- Node.js (para dependencias)
- Git y GitHub

##  Funcionalidades
- Navegación entre pantallas
- Componentes reutilizables
- Estructura modular del proyecto
- Consumo y manejo de datos locales
